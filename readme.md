# This landing page is designed on a basis of a template kindly provided by Evhen Andrikanych. <br>

## Lesson: "Adaptive layout of an interesting site. Explanation of actions. HTML CSS JS FIGMA." <br>

**_LAYOUT FEATURES:_**

- layout made on Figma
- HTML using BEM methodology
- no use of any preprocessors and assemblers (manual work)
- almost pixel perfect on the PC version :)
- fully responsive layout up to 360px width
- page width limited to max 1600px
- decreasing header when scrolling
- hamburger menu on mobile devices
- fullscreen block
- slider Swiper 
- accordion spoiler
- tabs
- smooth scrolling to a section from a menu item
- scroll up when clicking button in footer
- animation
- preloader
- contacts form
- custom numbered list
- popup window with embedded video
- popup gallery
